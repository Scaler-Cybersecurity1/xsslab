# XSS Vulnerable Training Website

This project is an intentionally vulnerable website for learning Cross-Site Scripting (XSS).

## GitHub Pages version

GitHub Pages serves the static files in this repo:

- `index.html`
- `lab-reflected.html`
- `lab-stored.html`
- `lab-dom.html`

If your repo is `https://scaler-cybersecurity1.github.io/xsslab/`, these links should work:

- [Home](https://scaler-cybersecurity1.github.io/xsslab/)
- [Lab 1 Reflected](https://scaler-cybersecurity1.github.io/xsslab/lab-reflected.html)
- [Lab 2 Stored](https://scaler-cybersecurity1.github.io/xsslab/lab-stored.html)
- [Lab 3 DOM](https://scaler-cybersecurity1.github.io/xsslab/lab-dom.html)

## Local Node version (optional)

```bash
npm install
npm start
```

Open: [http://localhost:3000](http://localhost:3000)

## Success Codes

- Reflected: `XSS-REFLECTED-7A91`
- Stored: `XSS-STORED-3F22`
- DOM: `XSS-DOM-9BC0`
